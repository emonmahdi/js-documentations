JavaScript কী?

JavaScript মূলত তৈরি করা হয়েছিল “ওয়েব পেজকে জীবন্ত করার জন্য”।

এই ভাষায় লেখা প্রোগ্রামগুলোকে বলা হয় স্ক্রিপ্ট (scripts)। এগুলো সরাসরি একটি ওয়েব পেজের HTML ফাইলের ভেতর লেখা যায় এবং পেজ লোড হওয়ার সাথে সাথেই স্বয়ংক্রিয়ভাবে চালু হয়।

স্ক্রিপ্টগুলো সাধারণ টেক্সট আকারে লেখা ও চালানো হয়। এগুলো চালানোর জন্য কোনো বিশেষ প্রস্তুতি বা কম্পাইল করার প্রয়োজন হয় না।

এই দিক থেকে JavaScript, Java নামের আরেকটি ভাষা থেকে সম্পূর্ণ আলাদা।



কেন এর নাম JavaScript?

যখন JavaScript তৈরি করা হয়েছিল, তখন এর নাম ছিল “LiveScript”। কিন্তু সেই সময় Java খুবই জনপ্রিয় ছিল, তাই নতুন এই ভাষাটিকে Java-এর “ছোট ভাই” হিসেবে উপস্থাপন করলে জনপ্রিয়তা পাবে—এই চিন্তা থেকেই এর নাম রাখা হয় JavaScript।

কিন্তু সময়ের সাথে সাথে JavaScript একটি সম্পূর্ণ স্বাধীন ভাষা হয়ে ওঠে, যার নিজস্ব স্পেসিফিকেশন রয়েছে, যাকে বলা হয় ECMAScript। বর্তমানে JavaScript-এর সাথে Java-এর কোনো সম্পর্ক নেই।



আজকের দিনে JavaScript শুধু ব্রাউজারেই নয়, বরং সার্ভারেও এবং এমনকি যেকোনো ডিভাইসেও চালানো যায়, যেখানে একটি বিশেষ প্রোগ্রাম থাকে যাকে বলা হয় JavaScript engine।

ব্রাউজারের ভেতরে একটি এমবেডেড (ভিতরে থাকা) ইঞ্জিন থাকে, যাকে অনেক সময় “JavaScript virtual machine” বলা হয়।

ভিন্ন ভিন্ন ব্রাউজারে ভিন্ন ভিন্ন JavaScript engine ব্যবহার করা হয়, যাদের আলাদা আলাদা কোডনেম আছে। যেমনঃ

V8 – Chrome, Opera এবং Edge ব্রাউজারে ব্যবহৃত হয়
SpiderMonkey – Firefox ব্রাউজারে ব্যবহৃত হয়
এছাড়াও আরও কিছু ইঞ্জিন আছে যেমনঃ
Chakra – Internet Explorer (IE)-এ
JavaScriptCore, Nitro, SquirrelFish – Safari ব্রাউজারে 



JavaScript Engine কীভাবে কাজ করে?

JavaScript ইঞ্জিনগুলো বেশ জটিল। তবে এর মূল ধারণাগুলো খুবই সহজ।

1. প্রথমে ইঞ্জিনটি স্ক্রিপ্টটি পড়ে (এটাকে বলা হয় parse করা)
2. এরপর স্ক্রিপ্টটিকে মেশিন কোডে রূপান্তর (compile) করে
3. তারপর সেই মেশিন কোডটি খুব দ্রুত গতিতে রান করে

এই পুরো প্রক্রিয়ার প্রতিটি ধাপেই ইঞ্জিন বিভিন্ন ধরনের অপ্টিমাইজেশন ব্যবহার করে।

এমনকি ইঞ্জিনটি স্ক্রিপ্ট চলার সময়ও সেটিকে পর্যবেক্ষণ করে, ডেটার প্রবাহ বিশ্লেষণ করে, এবং সেই তথ্য ব্যবহার করে মেশিন কোডকে আরও বেশি অপ্টিমাইজ করে।



ব্রাউজারের ভেতরে JavaScript কী কী করতে পারে?

In-browser JavaScript ওয়েবপেজের সাথে সম্পর্কিত প্রায় সব কাজই করতে পারে — যেমন ইউজারের সাথে ইন্টারঅ্যাকশন, পেজ ম্যানিপুলেশন এবং ওয়েব সার্ভারের সাথে যোগাযোগ।

উদাহরণ হিসেবে, ব্রাউজারের JavaScript পারে:

পেজে নতুন HTML যোগ করা, বিদ্যমান কনটেন্ট পরিবর্তন করা, স্টাইল (CSS) বদলানো
ইউজারের কাজ অনুযায়ী রেসপন্স দেওয়া (মাউস ক্লিক, মাউস নড়াচড়া, কিবোর্ড প্রেস ইত্যাদি)
রিমোট সার্ভারে নেটওয়ার্ক রিকোয়েস্ট পাঠানো, ফাইল ডাউনলোড ও আপলোড করা (AJAX এবং COMET প্রযুক্তি ব্যবহার করে)
কুকিজ সেট ও পড়া, ইউজারকে প্রশ্ন করা, মেসেজ দেখানো
ক্লায়েন্ট সাইডে ডেটা সংরক্ষণ করা (Local Storage)
