💡 JavaScript Interview Questions & Answers

1. What is the difference between var, let, and const?
  • var – Function scoped, can be re-declared and updated.
  • let – Block scoped, can be updated but not re-declared.
  • const – Block scoped, cannot be updated or re-declared.
var x = 10;
              let y = 20;
              const z = 30;
              
              x = 15; // OK
              y = 25; // OK
              z = 35; // ❌ Error: Assignment to constant variable
2. Difference between forEach, for, and map?
  • for: Traditional loop, flexible with break/continue.
  • forEach: Array method, no break/continue, doesn't return new array.
  • map: Returns new array, good for transformation.
const numbers = [1, 2, 3, 4];
              
              // for
              for (let i = 0; i < numbers.length; i++) {
                console.log(numbers[i]);
              }
              
              // forEach
              numbers.forEach(num => console.log(num));
              
              // map
              const doubled = numbers.map(num => num * 2);
              console.log(doubled); // [2, 4, 6, 8]
3. ES6 Features?
  • let & const
  • Arrow functions
  • Template literals
  • Destructuring
  • Default parameters
  • Spread/Rest operators
  • Promises & async/await
  • Modules, Classes
// Template literal
              const name = "Emon";
              console.log(`Hello, ${name}`);
              
              // Arrow function
              const add = (a, b) => a + b;
              
              // Destructuring
              const [x, y] = [10, 20];
              console.log(x, y); // 10 20
4. Difference between callback, promise, and async/await?
  • Callback: Function passed as an argument.
  • Promise: Represents future result, avoids callback hell.
  • Async/Await: Cleaner syntax to handle Promises.
// Callback
              function fetchData(callback) {
                setTimeout(() => callback("Done!"), 1000);
              }
              
              // Promise
              function fetchPromise() {
                return new Promise(resolve => {
                  setTimeout(() => resolve("Resolved!"), 1000);
                });
              }
              
              // Async/Await
              async function loadData() {
                const data = await fetchPromise();
                console.log(data); // "Resolved!"
              }
