# ЁЯФ╣ 3) **Memoization**

Expensive function-ржПрж░ result cache ржХрж░рзЗ ржкрж░рзЗрж░ ржмрж╛рж░ calculation ржирж╛ ржХрж░рзЗ cache ржерзЗржХрзЗ return ржХрж░рж╛ред

### тЬФ Example

```js
function memoize(fn) {
  let cache = {};
  return function (num) {
    if (cache[num]) return cache[num];
    const result = fn(num);
    cache[num] = result;
    return result;
  };
}

function slowSquare(n) {
  for (let i = 0; i < 100000000; i++) {}
  return n * n;
}

const fastSquare = memoize(slowSquare);

console.log(fastSquare(5)); // Fast on second call
console.log(fastSquare(5));
```

# ЁЯФ╣ 4) **Debounce (Most Asked)**

### тЬФ Example

```js

  return function (...args) {
```

# ЁЯФ╣ 5) **Throttle**

Throttle тЖТ ржирж┐рж░рзНржжрж┐рж╖рзНржЯ рж╕ржорзЯ ржкрж░ржкрж░ function execute рж╣рждрзЗ ржжрзЗрзЯред

ЁЯСЙ Used in: scroll event, window resize, mouse move tracking

### тЬФ Example

```js
function throttle(fn, delay) {
  let flag = true;

  return function (...args) {
    if (!flag) return;
    flag = false;
    fn.apply(this, args);
    setTimeout(() => (flag = true), delay);
  };
}
```
