# ðŸ”¹ 3) **FizzBuzz**

* Multiple of 3 â†’ "Fizz"
* Multiple of 5 â†’ "Buzz"

```js
function fizzBuzz(n) {
  for (let i = 1; i <= n; i++) {
    if (i % 15 === 0) console.log("FizzBuzz");
```

# ðŸ”¹ 4) **Find Largest / Smallest Number**

# ðŸ”¹ 5) **Check Palindrome**

```js
function isPalindrome(str) {
  const reversed = str.split("").reverse().join("");
  return str === reversed;
}

console.log(isPalindrome("madam")); // true
```

# ðŸ”¹ 6) **Count Vowels in String**

```js
function countVowels(str) {
  const vowels = "aeiouAEIOU";
  return [...str].filter(ch => vowels.includes(ch)).length;
}

console.log(countVowels("Emon")); // 2
```

# ðŸ”¹ 10) **Frequency Counter**

Count frequency of elements in array.

```js
function frequencyCounter(arr) {
  return arr.reduce((acc, curr) => {
    acc[curr] = (acc[curr] || 0) + 1;
    return acc;
  }, {});
}

console.log(frequencyCounter(["a", "b", "a"])); // { a: 2, b: 1 }
```

# ðŸ”¹ 11) **Anagram Check**

Check if two strings are anagrams.

```js
function isAnagram(str1, str2) {
  const normalize = str =>
    str.toLowerCase().split("").sort().join("");
  return normalize(str1) === normalize(str2);
}

console.log(isAnagram("listen", "silent")); // true
```
