JavaScript DOM Methods & Properties – Definitions with Examples

1. createElement()

Definition: Creates a new HTML element but does not place it in the DOM automatically.


const newDiv = document.createElement('div');
newDiv.textContent = 'This is a new div';
document.body.appendChild(newDiv);

2. appendChild()

Definition: Appends a node as the last child of a specified parent node.


const ul = document.createElement('ul');
const li = document.createElement('li');
li.textContent = 'Item 1';
ul.appendChild(li);
document.body.appendChild(ul);

3.textContent

Definition: Gets or sets the text content of a node, ignoring any HTML tags.


const para = document.createElement('p');
para.textContent = 'Hello, world!';
document.body.appendChild(para);

4. innerHTML

Definition: Gets or sets the HTML content (including tags) inside an element.


const container = document.createElement('div');
container.innerHTML = '<strong>Bold Text</strong>';
document.body.appendChild(container);

5. innerHTML vs. createElement()

Definition: innerHTML is faster but less secure. createElement() is safer and better for creating dynamic elements.


// Using innerHTML
const div1 = document.createElement('div');
div1.innerHTML = '<span>Hello HTML</span>';
document.body.appendChild(div1);

// Using createElement
const div2 = document.createElement('div');
const span = document.createElement('span');
span.textContent = 'Hello JS';
div2.appendChild(span);
document.body.appendChild(div2);

6. DocumentFragment

Definition: A lightweight container for DOM nodes, used to batch-insert multiple nodes efficiently.


const fragment = document.createDocumentFragment();
for (let i = 0; i < 3; i++) {
  const li = document.createElement('li');
  li.textContent = `Item ${i + 1}`;
  fragment.appendChild(li);
}
const ul = document.createElement('ul');
ul.appendChild(fragment);
document.body.appendChild(ul);

7. after()

Definition: Inserts a node directly after the specified element.


const ref = document.createElement('div');
ref.textContent = 'Reference Node';
document.body.appendChild(ref);

const newEl = document.createElement('div');
newEl.textContent = 'Inserted After';
ref.after(newEl);

8. append()

Definition: Appends one or more nodes or strings to the end of a parent node.


const div = document.createElement('div');
div.append('Text using append()');
document.body.appendChild(div);

9. prepend()

Definition: Inserts one or more nodes or strings at the beginning of a parent node.


const div = document.createElement('div');
div.textContent = 'End';
div.prepend('Start - ');
document.body.appendChild(div);

10. insertAdjacentHTML()

Definition: Parses text as HTML and inserts it into the DOM at a specified position.


const target = document.createElement('div');
target.textContent = 'Container';
document.body.appendChild(target);

target.insertAdjacentHTML('beforeend', '<p>Inserted HTML</p>');

11. replaceChild()

Definition: Replaces one child node of a specified parent node with another.


const parent = document.createElement('div');
const oldChild = document.createElement('p');
oldChild.textContent = 'Old';
parent.appendChild(oldChild);

const newChild = document.createElement('p');
newChild.textContent = 'New';
parent.replaceChild(newChild, oldChild);
document.body.appendChild(parent);

12. cloneNode()

Definition: Creates a copy of a node. Set true to clone with children.


const original = document.createElement('div');
original.textContent = 'Original';
document.body.appendChild(original);

const clone = original.cloneNode(true);
document.body.appendChild(clone);

13. removeChild()

Definition: Removes a child node from its parent.


const container = document.createElement('div');
const child = document.createElement('p');
child.textContent = 'Remove Me';
container.appendChild(child);
document.body.appendChild(container);

container.removeChild(child);

14. insertBefore()

Definition: Inserts a node before a reference node as a child of a specified parent.


const ul = document.createElement('ul');
const li2 = document.createElement('li');
li2.textContent = 'Second';
ul.appendChild(li2);

const li1 = document.createElement('li');
li1.textContent = 'First';
ul.insertBefore(li1, li2);

document.body.appendChild(ul);

15. insertAfter() (Custom Helper)

Definition: Inserts a node after a reference node (not built-in; we define it).


function insertAfter(newNode, refNode) {
  refNode.parentNode.insertBefore(newNode, refNode.nextSibling);
}

const ref = document.createElement('p');
ref.textContent = 'Reference';
document.body.appendChild(ref);

const next = document.createElement('p');
next.textContent = 'Inserted After';

insertAfter(next, ref);
