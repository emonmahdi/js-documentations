HTML Attributes & DOM Properties

1. HTML Attributes vs DOM Object Properties

HTML Attributes are defined in the HTML and represent the initial values of elements.

DOM Object Properties are JavaScript representations of the elements that reflect the current state.

  • Attribute: Static value from HTML – <input value="Hello">
  • Property: Dynamic JS property – element.value

Changing a DOM property doesn’t update the HTML attribute (and vice versa).
